Understanding Integers

Integers form a bigger collection of numbers which contains whole numbers and negative numbers.

What are Integers?

  • Positive Integers: 1, 2, 3, 4, ... (Natural numbers)
  • Zero: 0 is an integer which is neither positive nor negative.
  • Negative Integers: -1, -2, -3, -4, ...

The collection of integers is ..., -3, -2, -1, 0, 1, 2, 3, ...

Ordering of Integers

On a number line, numbers to the right are greater than numbers to the left.

Example: 5 > 2, 2 > 0, 0 > -1, -2 > -5.

Addition and Subtraction

Addition Rules:

  • Same signs: Add and keep the sign. (e.g., -2 + (-3) = -5)
  • Different signs: Subtract smaller absolute value from larger, keep sign of larger. (e.g., -5 + 3 = -2)

Subtraction Rule:

  • Subtracting a number is same as adding its additive inverse. (e.g., 5 - (-2) = 5 + 2 = 7)

Solved Examples

Example 1: Comparing Integers

Question: Which is greater: -15 or -10?

Solution:

On the number line, -10 is to the right of -15.

Therefore, -10 > -15.

Example 2: Adding Integers

Question: Solve: (-10) + (+3)

Solution:

We are adding integers with different signs.

Difference: 10 - 3 = 7.

Sign of larger number (10) is negative.

Answer: -7
