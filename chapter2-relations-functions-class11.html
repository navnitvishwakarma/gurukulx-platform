Cartesian Product

If A and B are two non-empty sets, then Cartesian product A × B is the set of all ordered pairs (a, b) such that a ∈ A and b ∈ B.

Relation

A relation R from set A to set B is a subset of A × B.

Domain: Set of all first elements of ordered pairs in R.

Range: Set of all second elements of ordered pairs in R.

Function

A relation f from A to B is a function if every element of set A has one and only one image in set B.

Solved Examples

Example 1: Cartesian Product

Question: A={1,2}, B={3,4}. Find A × B.

Solution: {(1,3), (1,4), (2,3), (2,4)}.

Example 2: Range of Function

Question: f(x) = x², x ∈ {-1, 0, 1}

Solution: f(-1)=1, f(0)=0, f(1)=1. Range = {0, 1}.
