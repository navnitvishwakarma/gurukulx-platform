Recording data, pictographs, and bar graphs

Understanding Data

Data is a collection of numbers gathered to give some information.

Recording Data

We often record data using tally marks. Tally marks are vertical lines used to keep count. Every 5th mark is a diagonal line across the previous 4 marks.

Pictographs

A pictograph represents data through pictures of objects. It helps answer questions near the data at a glance.

Note: Pictures can represent multiple units (e.g., 1 tree symbol = 10 actual trees).

Bar Graphs

Representing data using bars of uniform width. The height of the bar represents the frequency of the data item.

Solved Examples

Example 2: Reading Graphs

Question: If a bar graph shows sales of 50 units for Monday and 30 for Tuesday, which day had higher sales?

Solution:

Monday (50) > Tuesday (30).

So, Monday had higher sales.
